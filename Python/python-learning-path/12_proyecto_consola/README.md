# 12 - Proyecto Final: Aplicación de Consola

## Concepto
Crearás una aplicación de consola que integra varios conceptos aprendidos.

## Requisitos
- Menú interactivo
- Uso de funciones
- Manejo de errores
- Lectura/escritura de archivos

## Analogía sencilla
Piensa en una agenda digital: puedes agregar, ver y eliminar contactos desde la consola.

## Ejemplo explicado
La aplicación será una agenda de contactos:
- Agregar contacto
- Ver contactos
- Eliminar contacto

## Buenas prácticas
- Modulariza el código
- Maneja errores
- Usa nombres descriptivos

## Ejercicios prácticos
1. Implementa el menú interactivo.
2. Guarda los contactos en un archivo.

## Mini reto
Agrega una opción para buscar contactos por nombre.
