# 00 - Introducción a Python

## ¿Qué es Python?
Python es un lenguaje de programación fácil de aprender, muy utilizado en ciencia de datos, desarrollo web, automatización y más.

## ¿Para qué se usa Python?
- Desarrollo web
- Ciencia de datos
- Automatización de tareas
- Inteligencia artificial
- Juegos

## Analogía sencilla
Piensa en Python como una caja de herramientas: puedes construir desde una calculadora hasta una página web.

## Cómo instalar Python
- Ve a [python.org](https://www.python.org/downloads/)
- Descarga la versión recomendada
- Sigue las instrucciones según tu sistema operativo

## Cómo ejecutar scripts
1. Crea un archivo llamado `hola.py` con este contenido:
   ```python
   print("¡Hola, Python!")
   ```
2. Abre la terminal y navega a la carpeta donde está el archivo.
3. Ejecuta:
   ```bash
   python hola.py
   ```

## Uso básico de la terminal
- Navegar: `cd nombre_carpeta`
- Listar archivos: `ls`
- Ejecutar scripts: `python archivo.py`

## Ejemplo explicado
```python
# Este programa imprime un saludo
print("¡Hola, Python!")
```

## Errores comunes
- Olvidar guardar el archivo antes de ejecutar
- Escribir mal el nombre del archivo
- No tener Python instalado

## Buenas prácticas
- Usar nombres de archivos descriptivos
- Comentar el código
