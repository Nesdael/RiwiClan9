# 15 - Preparación Laboral Python Junior

## Concepto
Prepararse para el mercado laboral implica conocer las habilidades y prácticas que buscan las empresas.

## Analogía sencilla
Piensa en un atleta: entrena, mejora y muestra sus logros para ser seleccionado.

## Habilidades que pide el mercado
- Lógica de programación
- Manejo de estructuras de datos
- Uso de librerías estándar
- Programación orientada a objetos
- Manejo de archivos
- Buenas prácticas

## ¿Qué debe saber un Python junior?
- Escribir scripts funcionales
- Leer y escribir archivos
- Usar funciones y clases
- Manejar errores
- Trabajar en equipo

## Ejemplo de proyecto para portafolio
Una agenda de contactos en consola, con manejo de archivos y funciones.

## Tips para entrevistas
- Practica ejercicios de lógica
- Explica tu código con claridad
- Muestra proyectos propios
- Sé honesto sobre tus conocimientos

## Ejercicios prácticos
1. Haz una lista de tus habilidades técnicas.
2. Simula una entrevista respondiendo preguntas sobre tu proyecto.

## Mini reto
Prepara una presentación breve de tu agenda de contactos y explícalo como si estuvieras en una entrevista.
