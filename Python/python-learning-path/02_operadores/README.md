# 02 - Operadores en Python

## Concepto
Los operadores permiten realizar operaciones entre variables y valores.

## Analogía sencilla
Piensa en los operadores como herramientas matemáticas y lógicas: suman, comparan o combinan valores.

## Tipos de operadores
- Matemáticos: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- Comparación: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Lógicos: `and`, `or`, `not`

## Ejemplo explicado
```python
# Operadores matemáticos
suma = 2 + 3      # 5
resta = 5 - 2     # 3
multiplicacion = 4 * 2  # 8
division = 10 / 2 # 5.0
modulo = 7 % 3    # 1
potencia = 2 ** 3 # 8

# Operadores de comparación
print(5 == 5)     # True
print(3 != 2)     # True
print(4 > 2)      # True
print(2 < 1)      # False

# Operadores lógicos
print(True and False) # False
print(True or False)  # True
print(not True)       # False
```

## Errores comunes
- Usar `=` en vez de `==` para comparar
- Dividir por cero
- Confundir `and` con `&`

## Buenas prácticas
- Usar paréntesis para claridad
- Comentar operaciones complejas
