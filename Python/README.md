# Guía Completa de Aprendizaje Python

Bienvenido a **python-learning-path**, un repositorio educativo diseñado para personas con poca experiencia en programación. Aquí encontrarás una ruta de aprendizaje progresiva para adquirir las competencias necesarias y aplicar a ofertas junior de Python.

## Estructura de la guía
Cada carpeta representa un módulo de aprendizaje, con un README.md explicativo y ejemplos funcionales. El contenido avanza desde lo más básico hasta temas intermedios/avanzados.

### Módulos incluidos
- **00_introduccion**: Qué es Python, instalación y primeros pasos.
- **01_variables_y_tipos**: Variables, tipos de datos y ejemplos.
- **02_operadores**: Operadores matemáticos, lógicos y de comparación.
- **03_condicionales**: Estructuras if, elif, else.
- **04_ciclos**: Uso de for, while, break y continue.
- **05_estructuras_de_datos**: Listas, tuplas, diccionarios y sets.
- **06_funciones**: Definición, parámetros, retorno y docstrings.
- **07_manejo_de_errores**: try, except, finally y errores comunes.
- **08_modularizacion**: Importar módulos y organización de proyectos.
- **09_librerias_estandar**: Uso de librerías como random, datetime, math y requests.
- **10_manejo_de_archivos**: Lectura, escritura y manejo de CSV.
- **11_programacion_orientada_a_objetos**: Clases, atributos, métodos y encapsulación.
- **12_proyecto_consola**: Proyecto final integrador con menú interactivo.
- **13_introduccion_data_analysis**: Análisis de datos con pandas.
- **14_buenas_practicas**: PEP8, nombres de variables y estructura de proyectos.
- **15_preparacion_laboral**: Habilidades, tips y ejemplos para entrevistas.

## ¿Cómo usar la guía?
1. Lee el README.md de cada módulo.
2. Ejecuta y modifica los ejemplos de código.
3. Resuelve los ejercicios y mini retos.
4. Avanza módulo a módulo hasta completar el proyecto final.

## Recomendaciones
- Dedica tiempo a cada ejercicio.
- Consulta la documentación oficial de Python.
- Usa la terminal para ejecutar tus scripts.
- Comparte tus avances y dudas en comunidades de aprendizaje.

¡Éxito en tu camino para convertirte en desarrollador Python junior!
